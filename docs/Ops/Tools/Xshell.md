# Xshell的安装与使用

Xshell是一款用于在本地计算机上远程连接到服务器，并进行命令操作，文件管理等功能的SSH客户端软件。

**在Xshell中编写Linux命令和在Linux系统中的终端上编写命令是一样的。**
